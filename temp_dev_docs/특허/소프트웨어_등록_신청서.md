### 7.2 설치 방법

#### 7.2.1 Docker 설치
```bash
# 1. Docker 이미지 다운로드
docker pull ahp-platform:latest

# 2. 환경 변수 설정
cp .env.example .env
# .env 파일 편집

# 3. 컨테이너 실행
docker-compose up -d

# 4. 데이터베이스 초기화
docker exec ahp-platform python manage.py migrate

# 5. 관리자 계정 생성
docker exec -it ahp-platform python manage.py createsuperuser
```

#### 7.2.2 수동 설치
```bash
# 1. 소스코드 클론
git clone https://github.com/company/ahp-platform

# 2. 의존성 설치
cd frontend && npm install
cd ../backend && pip install -r requirements.txt

# 3. 데이터베이스 설정
createdb ahp_platform
python manage.py migrate

# 4. 빌드 및 실행
npm run build
python manage.py runserver
```
