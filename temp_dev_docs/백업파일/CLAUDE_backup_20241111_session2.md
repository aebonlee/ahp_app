### 설계 문서 활용
- 모든 구현은 Opus 설계 문서 기반
- API 엔드포인트는 설계 문서 참조
- 데이터 모델은 설계 문서의 스키마 사용
- 알고리즘은 설계 문서의 의사코드 참조

- TypeScript strict 모드 유지
