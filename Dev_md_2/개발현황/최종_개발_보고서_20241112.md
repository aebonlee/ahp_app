## 🆕 신규 그룹 평가 시스템 구현

### 구현한 핵심 구성요소

#### 1. TypeScript 타입 시스템
**파일**: `src/types/group.ts` (260줄)
**내용**:
- 15개 핵심 인터페이스 정의
- AIJ/AIP/Fuzzy 통합 방법론 지원
- Delphi 방법론 완전 지원
- WebSocket 실시간 통신 타입
- 컴포넌트 Props 및 API 응답 타입

```typescript
// 주요 인터페이스들
interface EvaluationGroup { ... }        // 평가 그룹
interface GroupMember { ... }            // 그룹 멤버  
interface ConsensusMetrics { ... }       // 합의 지표
interface DelphiRound { ... }           // Delphi 라운드
interface GroupCalculationResult { ... } // 통합 계산 결과
```

#### 2. 고급 계산 엔진
**파일**: `src/utils/groupCalculations.ts` (409줄)
**내용**:
- AIJ (기하평균 기반 개별 판단 통합) 완전 구현
- AIP (개별 우선순위 통합) 완전 구현  
- Shannon Entropy 기반 합의도 측정
- Kendall's W 일치계수 계산
- 이견 분석 및 해결 제안 시스템
- 품질 기반 가중치 자동 조정

```typescript
// 핵심 함수들
calculateAIJ(): AIJResult                    // 기하평균 통합
calculateAIP(): AIPResult                    // 우선순위 통합
calculateKendallsW(): number                 // 일치계수
analyzeDisagreements(): DisagreementAnalysis[] // 이견 분석
calculateGroupResult(): GroupCalculationResult // 종합 통합
```

#### 3. React 대시보드 컴포넌트
**파일**: `src/components/evaluation/group/GroupEvaluationDashboard.tsx` (622줄)
**내용**:
- 그룹 생성/참여/관리 UI
- 실시간 WebSocket 기반 모니터링
- 동기/비동기/Delphi 세션 지원
- 합의도 시각화 및 진행률 추적
- 참여자 활동 모니터링

#### 4. 완전한 API 서비스
**파일**: `src/services/groupApi.ts` (579줄)
**내용**:
- 40+ API 함수 완전 구현
- 그룹 관리 (생성, 수정, 삭제, 참여)
- 세션 관리 (동기/비동기/Delphi)
- 계산 및 통합 (AIJ/AIP/가중평균)
- 실시간 모니터링 및 WebSocket 연결
- 결과 내보내기 (Excel/PDF/JSON)
