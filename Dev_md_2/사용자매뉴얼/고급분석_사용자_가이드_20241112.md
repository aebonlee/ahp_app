## 🎯 고급 분석 시스템 소개

AHP 플랫폼의 고급 분석 시스템은 의사결정 결과에 대한 심층적인 통찰력을 제공합니다. 민감도 분석, 시나리오 분석, AI 기반 해석을 통해 더욱 신뢰할 수 있는 의사결정을 지원합니다.

### 주요 기능
- 🔍 **민감도 분석**: 기준 가중치 변화에 따른 순위 변동 분석
- 🎲 **시나리오 분석**: What-if 시뮬레이션과 목표 탐색
- 🤖 **AI 인사이트**: 자동 패턴 감지 및 개선 제안
- 📊 **종합 대시보드**: 모든 분석 결과의 통합 관리

---

## 🚀 시작하기

### 1. 고급 분석 대시보드 접근

1. AHP 프로젝트 메인 페이지에서 **"고급 분석"** 탭 클릭
2. 또는 프로젝트 완료 후 **"결과 분석"** → **"고급 분석"** 선택

### 2. 대시보드 구성

고급 분석 대시보드는 4개의 주요 탭으로 구성됩니다:

```
📋 개요        → 전체 분석 상태 및 요약
🔍 민감도 분석  → 토네이도 다이어그램 및 안정성 분석  
🎯 시나리오    → What-if 분석 및 목표 탐색
🤖 AI 인사이트 → 자동 결과 해석 및 추천사항
```

## 🔍 민감도 분석 사용법

민감도 분석은 각 기준의 가중치가 변할 때 대안 순위가 어떻게 변화하는지 분석합니다.

### 분석 실행

1. **"민감도 분석"** 탭 선택
2. **분석 대상 설정**:
   - **대안 선택**: 분석할 대안 선택
   - **기준 선택**: 특정 기준에 집중 (선택사항)
   - **분석 해상도**: 50단계(낮음) ~ 200단계(높음)

3. **"분석 실행"** 버튼 클릭

### 결과 해석

#### 📊 전체 안정성 지표
- **높음 (70% 이상)**: 가중치 변화에도 순위가 안정적
- **보통 (30-70%)**: 일부 가중치 변화 시 순위 변동 가능
- **낮음 (30% 미만)**: 가중치에 매우 민감, 신중한 검토 필요

#### 🌪️ 토네이도 다이어그램
- **가로축**: 대안의 점수 변화 범위
- **세로축**: 기준별 영향도 (위쪽이 더 민감)
- **색상**: 빨간색(감소), 초록색(증가)

**해석 예시**:
```
기준 A ████████████████████ (영향도 높음)
기준 B ██████████           (영향도 보통)  
기준 C ████                 (영향도 낮음)
```

#### ⚠️ 임계점 (Critical Points)
- **임계점**: 순위가 바뀌는 가중치 값
- **주의사항**: 임계점 근처에서는 작은 변화도 큰 영향

### 실용적 활용법

1. **가중치 검토**: 민감도가 높은 기준의 가중치를 재검토
2. **추가 분석**: 임계점 근처 기준들에 대해 추가 정보 수집
3. **의사결정 신뢰도**: 안정성이 높은 대안 선택 고려

## 🎯 시나리오 분석 사용법

시나리오 분석을 통해 다양한 가정 하에서의 결과를 예측할 수 있습니다.

### What-if 분석

#### 1. 새 시나리오 생성

1. **"시나리오 분석"** 탭 → **"시나리오 생성"** 서브탭
2. **시나리오 정보 입력**:
   - **이름**: 예) "경제 위기 상황"
   - **설명**: 시나리오의 배경 설명
   - **유형**: What-if 분석, 스트레스 테스트, 최적화

#### 2. 변경사항 정의

**변경 유형**:
- **가중치**: 기준의 중요도 변경
- **성능 점수**: 대안의 성능 값 변경
- **쌍대비교**: 비교 판단 변경

**예시 시나리오**:
```
📊 시나리오: "비용 절감 필요 상황"
├── 비용 기준 가중치: 0.3 → 0.5 (+67%)
├── 품질 기준 가중치: 0.4 → 0.3 (-25%)
└── 변경 이유: 예산 삭감으로 비용 중시
```

#### 3. 결과 분석

- **순위 변화**: 원래 순위 vs 시나리오 순위
- **영향도**: 경미/보통/주요 변화로 분류
- **신뢰도**: 시나리오 결과의 확실성 수준

### 목표 탐색 (Goal Seeking)

특정 대안을 원하는 순위로 만들기 위한 조건을 찾습니다.

#### 사용법

1. **"목표 최적화"** 서브탭 선택
2. **목표 설정**:
   - **목표 대안**: 순위를 올리고 싶은 대안
   - **목표 순위**: 원하는 순위 (예: 1순위)

3. **"목표 탐색 실행"** 클릭

#### 결과 해석

**성공 사례**:
```
✅ 목표 달성 가능!
실현 가능성: 85%
필요한 변경사항: 3개
- 기준 A 가중치: 0.3 → 0.4
- 기준 B 가중치: 0.2 → 0.15
- 대안 X 성능: 7점 → 8점
```

**실패 사례**:
```
❌ 목표 달성 어려움
현재 순위: 3순위, 목표 순위: 1순위
권장사항: 대안의 근본적 성능 개선 필요
```

### 시나리오 비교

여러 시나리오를 동시에 비교할 수 있습니다.

1. 비교할 시나리오들 체크박스 선택 (2개 이상)
2. **"시나리오 비교 실행"** 클릭
3. 상호작용 효과 및 시너지 분석 확인

## 🤖 AI 인사이트 활용법

AI 시스템이 분석 결과를 자동으로 해석하고 실용적인 조언을 제공합니다.

### AI 인사이트 유형

#### 🔍 패턴 감지
- **일관된 순위**: 여러 분석에서 동일한 결과
- **변동 패턴**: 특정 조건에서 반복되는 변화
- **성능 트렌드**: 시간에 따른 성능 변화

#### ⚠️ 이상 징후
- **예상 외 결과**: 일반적 기대와 다른 순위
- **불안정성**: 과도한 민감도 또는 변동성
- **일관성 문제**: 평가 판단의 논리적 모순

#### 🎯 개선 기회
- **성능 향상**: 대안별 개선 가능 영역
- **가중치 최적화**: 더 나은 의사결정을 위한 가중치 조정
- **평가 품질**: 평가 과정의 개선 방안

#### 🛡️ 위험 요소
- **의사결정 리스크**: 불확실성이 높은 영역
- **민감도 위험**: 작은 변화가 큰 영향을 미치는 부분
- **편향 가능성**: 평가 과정의 잠재적 편향

### AI 권고사항 활용

#### 권고사항 유형
- **🚨 긴급**: 즉시 조치가 필요한 사항
- **📋 전략적**: 중장기적 개선 방안
- **🛡️ 예방적**: 잠재적 문제 방지 방안
- **⚙️ 최적화**: 성능 향상 방안

#### 실행 우선순위
1. **높음**: 의사결정 신뢰도에 직접적 영향
2. **보통**: 장기적 품질 개선에 도움
3. **낮음**: 추가적 개선 효과

### AI 인사이트 신뢰도

AI 분석의 신뢰도는 5단계로 표시됩니다:

```
● ● ● ● ● 95% - 매우 높은 신뢰도
● ● ● ● ○ 80% - 높은 신뢰도  
● ● ● ○ ○ 60% - 보통 신뢰도
● ● ○ ○ ○ 40% - 낮은 신뢰도
● ○ ○ ○ ○ 20% - 매우 낮은 신뢰도
```

## 📊 종합 분석 실행

모든 분석을 한 번에 실행하여 종합적인 인사이트를 얻을 수 있습니다.

### 종합 분석 절차

1. **"개요"** 탭에서 **"종합 분석 실행"** 클릭
2. **분석 설정** (필요시):
   - 병렬 처리 활성화 (권장)
   - 최대 반복횟수 설정
   - 결과 캐싱 여부

3. **실시간 진행률 모니터링**:
   ```
   📊 민감도 분석    ✅ 완료
   🎯 시나리오 분석  🔄 진행중 (75%)
   🤖 AI 인사이트    ⏳ 대기
   📈 로버스트성     ⏳ 대기
   ```

4. **완료 후 종합 결과 확인**

### 결과 내보내기

분석 결과를 다양한 형태로 내보낼 수 있습니다:

#### PDF 리포트
- **경영진 보고용**: 요약 위주의 간결한 리포트
- **기술진 검토용**: 상세한 분석 과정 및 데이터 포함
- **사용자 정의**: 필요한 섹션만 선택하여 생성

#### Excel 워크북
- **원시 데이터**: 모든 계산 결과 및 중간 데이터
- **차트 및 그래프**: 시각화 자료 포함
- **피벗 테이블**: 동적 분석을 위한 데이터 구조

#### JSON 데이터
- **시스템 통합**: 다른 시스템과의 데이터 연동
- **추가 분석**: 외부 도구를 사용한 심화 분석

## 💡 실무 활용 팁

### 1. 분석 순서 권장사항

```
1️⃣ 기본 AHP 결과 확인 → 전체적인 순위 파악
2️⃣ 민감도 분석 실행 → 결과의 안정성 확인
3️⃣ 주요 시나리오 분석 → 다양한 상황 대비
4️⃣ AI 인사이트 검토 → 놓친 부분 발견
5️⃣ 종합적 의사결정 → 모든 정보 종합 판단
```

### 2. 의사결정 신뢰도 확인 체크리스트

- [ ] 전체 안정성 70% 이상
- [ ] 주요 시나리오에서 일관된 결과
- [ ] AI가 감지한 이상 징후 없음
- [ ] 임계점들이 현실적인 범위 내
- [ ] 모든 이해관계자가 중요하게 여기는 시나리오 검토 완료

### 3. 문제 해결 가이드

#### 안정성이 낮을 때
- 가중치 설정 과정 재검토
- 추가 전문가 의견 수렴
- 기준 정의의 명확성 확인

#### 예상과 다른 결과가 나올 때
- 입력 데이터 재확인
- 평가 과정의 편향 검토
- AI 인사이트의 이상 징후 분석

#### 시나리오 분석에서 큰 차이가 날 때
- 변화 요인의 현실성 검토
- 점진적 변화 시나리오 추가 분석
- 리스크 관리 방안 수립

## 🔧 고급 기능

### 1. 맞춤형 분석 설정

**분석 해상도 조정**:
- 빠른 분석: 50단계 (약 30초)
- 정밀 분석: 200단계 (약 2분)
- 초정밀 분석: 500단계 (약 5분)

**병렬 처리 최적화**:
- CPU 코어 수에 따른 자동 조정
- 메모리 사용량 모니터링
- 백그라운드 실행 지원

### 2. 협업 기능

**분석 결과 공유**:
- 팀원들과 실시간 분석 결과 공유
- 댓글 및 피드백 기능
- 버전 관리 및 변경 이력 추적

**권한 관리**:
- 보기 전용: 결과만 확인 가능
- 분석 실행: 새로운 분석 가능
- 완전 제어: 설정 변경 및 삭제 가능

### 3. 통합 연동

**외부 데이터 연동**:
- Excel/CSV 파일 가져오기
- 데이터베이스 실시간 연동
- API를 통한 자동 업데이트

**시스템 통합**:
- ERP 시스템 연동
- BI 도구 연결
- 보고서 자동 생성 및 배포
