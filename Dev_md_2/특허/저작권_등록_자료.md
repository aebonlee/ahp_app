## 2. 프로그램 설명

### 2.1 프로그램의 목적 및 기능

#### 목적
복잡한 의사결정 문제를 체계적으로 분석하고 해결하기 위한 웹 기반 플랫폼으로, AHP(Analytic Hierarchy Process) 방법론을 활용하여 다수의 평가자가 참여하는 그룹 의사결정을 지원

#### 주요 기능
1. **계층 구조 모델링**
   - 드래그 앤 드롭 방식의 시각적 모델 빌더
   - 다층 계층 구조 생성 및 관리
   - 기준/하위기준/대안 체계적 구성

2. **쌍대비교 평가**
   - 9점 척도 쌍대비교 매트릭스
   - 실시간 일관성 비율(CR) 계산
   - AI 기반 평가 가이드 제공

3. **다중 평가자 관리**
   - 이메일 초대 시스템
   - QR 코드 기반 익명 평가
   - 평가자별 진행률 추적

4. **분석 및 보고서**
   - 가중치 자동 계산
   - 민감도 분석
   - 그룹 의사결정 통합
   - PDF/Excel 보고서 생성

5. **실시간 협업**
   - WebSocket 기반 동시 편집
   - 실시간 데이터 동기화
   - 변경사항 즉시 반영

### 2.2 프로그램의 특징
- 클라우드 기반 SaaS 서비스
- 반응형 웹 디자인 (모바일/태블릿/PC)
- 다국어 지원 (한국어/영어)
- RESTful API 제공
- OAuth 2.0 소셜 로그인
